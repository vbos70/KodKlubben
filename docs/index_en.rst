Welcome to the Code Club
========================

There are several projects, called club_1, club_2, ...  They are
ordered from easiest to hardest.

The first one, club_1, you have to do. It is about setting up the
RaspberryPi, monitor, mouse, and keyboard. Once you have done that,
you can start using the RaspberryPi and start working on club_2,
club_3, ...
