Welcome to the Code Club
========================

There are several projects, called club 1, club 2, ... The first one, club_1,
you have to do. It is about setting up the RaspberryPi, monitor,
mouse, keyboard, etc. Once you have done that, you can start using the
RaspberryPi.

You can pick any of the other projects to work on. They are ordered
from easy (club_2) to hard (club_10).
