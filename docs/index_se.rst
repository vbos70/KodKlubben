Pong
----

:doc:`pong`

Spelet *Pong* är urgammal och kanska tråkigt. Men det funkar bra för
att lära programmera. Spelet är 2-dimensional och räknar med X och
Y-koordinater för att kontrollera bollen och spelarna.
